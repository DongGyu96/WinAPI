연습문제 3-3
신호등을 사용하여 이동하는 도형 그리기

화면을 y값으로 3등분 하고 각 등분에 임의의 크기의 사각형을 그린다.
각 도형의 이동속도는 다르다.
	맨 위의 사각형은 왼쪽에서 오른쪽으로 이동한다.
	중간 사각형은 오른쪽에서 왼쪽으로 이동한다.
	맨 아래의 사각형은 왼쪽에서 오른쪽으로 이동한다.
신호등 기능 : 화면의 파랑색/노랑색/빨강색의 신호등이 그려져있다. 아래의 키보드에 의해 
	      신호등이 선택된다.
	b/B : 이동 시작
	Y/y : 3초 후 정지
	r/R : 바로 정지
이동속도는 각각 다르게 설정한다.
왼쪽 또는 오른쪽 가장자리에 도달하면 반대편 가장자리에서 나타난다. 이 때 화면에
안 그려지는 일부분의 사각형을 반대편에 그리도록 한다.
